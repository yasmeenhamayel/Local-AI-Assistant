# Local-AI-Assistant
A robust Local AI Assistant built with Python and Streamlit, featuring session persistence and privacy focused design

Key Features
Session Persistence: Integrated with Streamlit's session_state to ensure chat history is preserved even after a page refresh.

Robust Handling: Specifically designed to handle irregular inputs (noise/gibberish) without crashing, maintaining a stable user experience.

Privacy-First: Operates entirely offline/locally, meaning no data is sent to external servers.

Real-time Response: Fast and responsive UI for a smooth conversation flow.

Tech Stack
Frontend: Streamlit
Language: Python 3.x

Process Handling: OS and Time libraries for efficient file and session management.
